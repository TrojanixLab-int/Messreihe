# ☢️>🖥️ MESSREIHE 🇺🇸 🇬🇧 🇩🇪 🇫🇷 🇪🇸 🇷🇺 Multi Language:

🇺🇸/🇬🇧 A portable multi-language radiation analysis and calculation software for Geiger-Müller tubes using audio-ADC or manual insertion of measurements, sequential filtering and an isotope library. It differentiates radiation types (Ambient Dose Rate, Alpha, Beta, Gamma). Available in English, French, Spanish, Russian and German. Read ReadMe!.txt for details or watch the Messreihe.pdf!
